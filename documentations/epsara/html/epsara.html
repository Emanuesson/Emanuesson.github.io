Epsara is a general-purpose physical analysis software, which provides a framework and programming library for all kinds of analytical questions. It was mainly designed for ion beam analysis, especially particle scattering spectrometry, but can be easily extended to most physcial analytical process, based on spectrometry. It works on many UNIX-like platforms, as well as Windows and OS X. Epsara is released under the GNU General Public License (GNU GPL).
