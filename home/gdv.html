LibGDV

Working in a scientific environment requires a fast, open and simple approach in software-design. The gtk+ library offers many of these features in terms of UI-programming and the underlying GLib is a powerful approach to use object-oriented programming (OOP) mostly independent of the available platform or programming language. However, most free scientific code does not apply Gtk+ even in applications were the integration would be highly preferable.
